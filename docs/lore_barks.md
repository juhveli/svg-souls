# Lore Barks: The Voice of the Shatter

## I. System Overview
See `active_storytelling.md` for technical implementation. This document contains the script.
*   **Context:** Enemies are trapped in time loops or delusions. Their dialogue should reflect their specific tragedy.
*   **Format:** `[State] Text`

## III. Boss Dialogue (The Dissonance Kings)

*   **[Phase 2 (Molten)]** "Burn the waste! Purify the scrap!"
